# 🛍️ ShopHub – Modern eCommerce Website

ShopHub is a responsive and modern eCommerce website designed to deliver a smooth online shopping experience. It includes product browsing, search, cart management, checkout flow, and user-friendly UI for customers.

## 🚀 Live Demo
[Visit Website](https://itsadarshpal.github.io/ShopHub-Ecommerce/)

---

## 📌 Features

- 🛒 Product Listing Page
- 🔍 Search & Filter Products
- ➕ Add to Cart / Remove from Cart
- 👀 Track Order
- 💳 Checkout Page
- 👤 User Login / Signup
- 📱 Fully Responsive Design
- ⚡ Fast & Clean User Interface
- 📦 Order Summary
- 🌙 Modern UI Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
